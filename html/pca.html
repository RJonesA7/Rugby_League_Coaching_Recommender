How to interpret a principal component

A principal component is not a single stat. It is a direction through the original data formed from weighted combinations of match variables. High or low values on a component indicate that several underlying statistics are tending to move together.

In coaching terms, this means you should not read “target PC1” as a literal instruction. Instead, treat it as: target the style of performance represented by the important loadings within that component.

Match statistics are first transformed into principal components. The predictive model operates on these components rather than the raw statistics. SHAP analysis then explains which components most often influenced the model’s predictions.

Reading a component

More often caused the model to predict defeat
More often caused the model to predict victory

A high score on a component means the combination of statistics that define that component were above their typical level in historical matches. If the SHAP value is positive, that pattern of statistics frequently pushed the model towards predicting a win. If negative, it tended to push the model towards predicting a loss.

Practical coaching interpretation

If a component has a high positive SHAP value, that doesn't suggest you should improve every variable inside it equally. It means the model found that opponents with that overall statistical pattern were associated with stronger predicted outcomes.

This gives a practical way of following the model's recommendations:

  1. Identify the components with the strongest positive SHAP values.
  2. Inspect the largest positive and negative loadings for that component.
  3. Translate those loadings into a rugby description such as yardage, ruck control, defensive pressure, or attacking fluency.
  4. Use that description as the coaching message, rather than the component label itself.

Example: if PC2 loads positively on all run metres, average set distance, and kick return metres, then a high value on PC2 may be interpreted as a territory and yardage profile. If that component is highly positive in SHAP terms, the recommendation is really to target that broader territorial style.

What the stat loadings table shows

Each principal component is built from the original match statistics using a set of weights called loadings. The loading value tells you how strongly a given statistic contributes to a given component.

A large positive loading means that higher values of that statistic push the team profile towards the positive end of that component. A large negative loading means that higher values of that statistic push the team profile towards the negative end of that component.

This table is therefore the direct bridge between abstract component labels such as PC1 and actual rugby variables such as all run metres, kicking metres, or completion rate.

In practice, once you identify an influential component from the SHAP output, inspect the strongest positive and negative loadings in its column to understand the kind of performance profile that component represents. Since principal components are formed from correlated variables, components are often recognisable as dimensions of a team's play. For instance, PC1 is often recognisable as a possession dominance axis, with positive loadings for time in possession and accordingly positive values for run metres and passes, but negative loadings for defensive workload stats like tackles made.
